  • 3、float(fl)

    默认使用float时,如需要IE兼容需将设置中的$lte7改为true,方可解决IE下浮动双倍间距的问题;

                        background-color: #DDD;
                        padding: 10px 0;
                        &-fl {
                          @include fl();
                          font-size: 16px;
                          background-color: #27AE60;
                          color: $F;
                        }
                      }
